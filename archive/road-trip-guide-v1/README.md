# Архив: road-trip-guide (первая версия)

Первая, полностью ручная версия проекта — статический HTML-путеводитель
по конкретному маршруту Кисловодск → Элиста → Волгоград → Самара
(позже Кисловодск–Элиста убрали, финальная опубликованная версия
охватывала Элиста → Самара).

Собрано вручную: веб-поиск + перекрёстная проверка через несколько
источников на каждую точку, аудио — через локальный `say` (macOS),
голос Milena. Заняло часы агентского времени на один маршрут — именно
это и стало поводом спроектировать Roadcast как сервис с другим,
намного более быстрым и дешёвым пайплайном (см. `../../ARCHITECTURE.md`,
принцип Wikipedia-first).

Живая версия (может быть уже неактуальна): https://bggrowthhack-optimizer.github.io/road-trip-guide/

## Что здесь ценного для нового проекта

- `PROCESS.md` — пройденный вручную пайплайн со всеми найденными
  ловушками (тёзки населённых пунктов, неверные административные
  привязки, точки в стороне от трассы) — стоит держать в голове при
  проектировании автоматической проверки данных
- `index.html` + `prototype.html` — рабочая механика подкаст-плеера
  (перемотка назад на 10 сек, переход между главами, автопереход) и
  формат вложенности достопримечательностей внутрь карточки населённого
  пункта — можно переносить в новый фронтенд как проверенный UX-паттерн
- `audio/*.m4a` — примеры финального аудио-рассказа (формат — связный
  монолог, не диалог; диалог пробовали и отклонили)

## Что НЕ переносить как есть
