# Roadcast

Сервис, который превращает любое путешествие в «книгу» — связную историю
мест по пути, с текстом и аудио-рассказом. Пользователь называет маршрут
(старт, финиш, опционально промежуточные точки) и способ передвижения —
сервис собирает историю сам.

Архитектура и план — в [ARCHITECTURE.md](ARCHITECTURE.md). Живые правила
разработки — в [CLAUDE.md](CLAUDE.md).

## Порядок сценариев

1. Автомобильное путешествие — первый MVP
2. Поезд
3. Прогулка по городу
4. Самолёт

## Структура репозитория

- `backend/` — FastAPI-сервис: маршрутизация запросов, оркестрация
  генерации контента, кеш точек
- `frontend/` — Next.js-приложение: форма ввода маршрута, читалка/плеер
- `docs/` — справочная документация по подсистемам (по мере появления)
- `archive/road-trip-guide-v1/` — первая версия проекта: полностью ручной
  статический HTML-гид по маршруту Кисловодск → Самара, собранный вручную
  как проверка идеи перед автоматизацией. Не рабочий код сервиса, но
  источник проверенных решений (формат аудио, механика плеера, структура
  вложенности) — см. пояснение в `CLAUDE.md`.

## Статус

Заложен фундамент: архитектура зафиксирована, идёт скаффолдинг
backend/frontend. Рабочего MVP ещё нет.
