# ИИ Путеводитель

Идея проекта: превратить автомобильное путешествие по России в приключение
с чек-поинтами. Пользователь называет только ключевые точки маршрута —
остальную цепочку населённых пунктов и достопримечательностей, а также
контент по каждой (история, факты, личности, аудио-рассказ), система
определяет и собирает сама. Повторяемый процесс описан в `PROCESS.md`.

Текущий маршрут — **Элиста → Волгоград → Камышин → Саратов → Хвалынск →
Сызрань → Самара**, одна цельная страница `index.html`.

Живая версия: https://bggrowthhack-optimizer.github.io/road-trip-guide/

Участок Кисловодск–Элиста, с которого начинался проект, из текущей версии
убран по решению пользователя после первой пробной поездки — материалы по
нему сохранены в `itogovyi-marshrut-svodnyi.md` на случай, если понадобятся
позже.

## Файлы

- `index.html` — путеводитель, один самодостаточный файл (открывается
  в браузере на телефоне без интернета, кроме загрузки аудио)
- `audio/*.m4a` — аудио-рассказы по каждой точке и почти каждой
  достопримечательности, голос встроенного macOS TTS (Milena)
- `itogovyi-marshrut-svodnyi.md` — исходный сводный маршрут для первой,
  уже не используемой версии (Кисловодск–Волгоград) — архив
- `PROCESS.md` — повторяемый пайплайн: как из пары ключевых точек
  получить полный маршрут и контент по нему
- `AGENT_PROMPT_TEMPLATE.md` — шаблон промпта для этапа генерации
  контента по точкам маршрута
- `prototype.html` — черновой прототип формата (вложенность, дистанция,
  подкаст-плеер), сохранён как референс истории решений

## Формат страницы

- Точки маршрута идут единой хронологической цепочкой по плечам,
  природные и рукотворные достопримечательности вложены в карточку
  ближайшего населённого пункта
- Бейдж приблизительной дистанции от старта (Элиста = 0 км) на каждой точке
- Флаг ⚠️ на карточках с оговорками — спорные даты, точки рядом с трассой,
  административные уточнения
- Один общий подкаст-плеер внизу экрана: play/пауза, перемотка назад на
  10 сек, переход между главами (главы — те же точки и достопримечательности,
  в порядке маршрута), автопереход по завершении

## Статус

Собрано на основе веб-исследования и перекрёстной проверки источников
(Википедия, официальные порталы, СМИ, реестры объектов культурного
наследия, грузоперевозочные сервисы для сверки списков населённых пунктов).

## Как посмотреть

Открыть `index.html` в браузере — двойным кликом локально или по ссылке
GitHub Pages выше.
