# ИИ Путеводитель

Идея проекта: превратить автомобильное путешествие по России в приключение
с чек-поинтами. Пользователь называет только ключевые точки маршрута —
остальную цепочку населённых пунктов и достопримечательностей, а также
контент по каждой (история, факты, личности), система определяет и
собирает сама. Повторяемый процесс описан в `PROCESS.md`.

Первый готовый пример — маршрут Кисловодск → Минеральные Воды →
Благодарный → Элиста → Волгоград.

## Файлы

- `index.html` — готовый мобильный путеводитель по первому маршруту,
  один самодостаточный файл (открывается в браузере на телефоне без
  интернета, без зависимостей)
- `itogovyi-marshrut-svodnyi.md` — исходный сводный маршрут с перечнем
  контрольных точек для этого примера
- `PROCESS.md` — повторяемый пайплайн: как из пары ключевых точек
  получить полный маршрут и контент по нему
- `AGENT_PROMPT_TEMPLATE.md` — шаблон промпта для этапа генерации
  контента по точкам маршрута

## Статус

Первая версия собрана и проверена на web-исследовании (Википедия,
официальные порталы, СМИ, реестры объектов культурного наследия).
Часть пунктов помечена флагом ⚠️ — спорные даты, административные
уточнения или недавние события, которые стоит перепроверить на месте.

Сейчас дорабатывается по итогам первой поездки (участок Кисловодск–Элиста).

## Как посмотреть

Просто открыть `index.html` в любом браузере — двойным кликом или
через GitHub Pages после публикации.
