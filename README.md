# Путеводитель: Кисловодск → Волгоград

Мобильный HTML-путеводитель по контрольным точкам автомобильного маршрута
Кисловодск → Минеральные Воды → Благодарный → Элиста → Волгоград.

По каждому пункту маршрута (населённые пункты, природные и рукотворные
достопримечательности) — краткая справка: история основания/открытия,
интересные факты, связанные личности.

## Файлы

- `index.html` — сам путеводитель, один самодостаточный файл (открывается
  в браузере на телефоне без интернета, без зависимостей)
- `itogovyi-marshrut-svodnyi.md` — исходный сводный маршрут с перечнем
  контрольных точек, на основе которого собран путеводитель

## Статус

Первая версия собрана и проверена на web-исследовании (Википедия,
официальные порталы, СМИ, реестры объектов культурного наследия).
Часть пунктов помечена флагом ⚠️ — спорные даты, административные
уточнения или недавние события, которые стоит перепроверить на месте.

Сейчас дорабатывается по итогам первой поездки (участок Кисловодск–Элиста).

## Как посмотреть

Просто открыть `index.html` в любом браузере — двойным кликом или
через GitHub Pages после публикации.
